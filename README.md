# KoishiRouterJS
VanillaJS Router 구현 연습 목적으로 만들어진 싱글 페이지 코이시 아티클 열람 앱   
릴리즈 예정

# Features (사용된 요소)
- `window.history` 객체를 사용한 SPA Router 기능.
- `Component` 클래스 정의 및 상속, 새로운 컴포넌트 생성 시 자동 render.
- `setState`를 통해 상태 변경 시 자동 재render.
